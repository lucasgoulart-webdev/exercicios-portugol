# exercicios-portugol
Programming logic exercises using Portugol

## Developers:
- Lucas Goulart Baumgratz

## Features
- Register up to 5 students (it could be more but the requirement was limitedd to 5)
- Store grades
- Calculate average
- Display report cards of each student after the grade registration

## Content
This repository contains basic programming logic exercises, including:
- Conditional structures
- Loops 
- Variables and basic operations

## Objective
Practice programming logic and improve problem-solving skills
